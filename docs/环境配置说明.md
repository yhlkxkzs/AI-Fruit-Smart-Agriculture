# 任务1: 水果种类识别 - 环境配置说明

## 📋 环境要求

### 基础要求
- **操作系统**: Linux / macOS / Windows
- **Python版本**: >= 3.8, < 3.11（推荐3.9或3.10）
- **CUDA**: 可选，推荐11.8或12.1（如果使用GPU）
- **内存**: 至少8GB（推荐16GB）
- **存储**: 至少10GB可用空间（用于数据集和模型）

## 🐍 Conda环境创建步骤

### 步骤1: 创建Conda环境

```bash

conda create -n afsa_task1 python=3.10 -y

# 激活环境
conda activate afsa_task1
```

### 步骤2: 安装PyTorch

**如果使用GPU（推荐）**:
```bash

conda install pytorch torchvision torchaudio pytorch-cuda=11.8 -c pytorch -c nvidia -y

# 或 CUDA 12.1版本
conda install pytorch torchvision torchaudio pytorch-cuda=12.1 -c pytorch -c nvidia -y
```

```bash
conda install pytorch torchvision torchaudio cpuonly -c pytorch -y
```

**验证PyTorch安装**:
```bash
python -c "import torch; print(f'PyTorch版本: {torch.__version__}'); print(f'CUDA可用: {torch.cuda.is_available()}')"
```

## ✅ 环境验证

创建文件 `verify_environment.py`:

```python
#!/usr/bin/env python3
"""验证环境配置是否正确"""

import sys

def check_package(package_name, import_name=None):
    """检查包是否安装"""
    if import_name is None:
        import_name = package_name
    
    try:
        __import__(import_name)
        print(f"✅ {package_name} 已安装")
        return True
    except ImportError:
        print(f"❌ {package_name} 未安装")
        return False

print("=" * 60)
print("环境配置验证")
print("=" * 60)

# 检查Python版本
python_version = sys.version_info
print(f"\nPython版本: {python_version.major}.{python_version.minor}.{python_version.micro}")
if python_version.major < 3 or (python_version.major == 3 and python_version.minor < 8):
    print("❌ Python版本过低，需要 >= 3.8")
else:
    print("✅ Python版本符合要求")

# 检查核心包
print("\n检查核心包:")
packages = [
    ('torch', 'torch'),
    ('torchvision', 'torchvision'),
    ('ultralytics', 'ultralytics'),
    ('cv2', 'opencv-python'),
    ('PIL', 'Pillow'),
    ('numpy', 'numpy'),
    ('pandas', 'pandas'),
]

all_ok = True
for import_name, package_name in packages:
    if not check_package(package_name, import_name):
        all_ok = False

# 检查PyTorch和CUDA
print("\n检查PyTorch配置:")
try:
    import torch
    print(f"  PyTorch版本: {torch.__version__}")
    print(f"  CUDA可用: {torch.cuda.is_available()}")
    if torch.cuda.is_available():
        print(f"  CUDA版本: {torch.version.cuda}")
        print(f"  GPU数量: {torch.cuda.device_count()}")
        for i in range(torch.cuda.device_count()):
            print(f"  GPU {i}: {torch.cuda.get_device_name(i)}")
except Exception as e:
    print(f"❌ PyTorch检查失败: {e}")
    all_ok = False

# 检查YOLOv8
print("\n检查YOLOv8:")
try:
    from ultralytics import YOLO
    print("✅ YOLOv8可以正常导入")
except Exception as e:
    print(f"❌ YOLOv8导入失败: {e}")
    all_ok = False

print("\n" + "=" * 60)
if all_ok:
    print("✅ 环境配置验证通过！")
else:
    print("❌ 环境配置存在问题，请检查上述错误")
print("=" * 60)
```

运行验证:
```bash
python verify_environment.py
```

## 🔧 常见问题

### 1. CUDA版本不匹配
**问题**: PyTorch无法使用GPU

**解决方案**:
- 检查CUDA版本: `nvidia-smi`
- 安装对应版本的PyTorch
- 或使用CPU版本

### 2. ultralytics安装失败
**问题**: pip install ultralytics失败

**解决方案**:
```bash
# 升级pip
pip install --upgrade pip

# 使用国内镜像
pip install ultralytics -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 3. 依赖冲突
**问题**: 包版本冲突

**解决方案**:
```bash
# 创建全新的conda环境
conda create -n afsa_task1_new python=3.10 -y
conda activate afsa_task1_new

# 按顺序安装
pip install torch torchvision torchaudio
pip install ultralytics
pip install opencv-python Pillow
```
